# Jun Alvior - Personal Portfolio

> **"Faster. Cleaner. Smarter digital experiences."**

A high-performance, responsive personal portfolio website designed to showcase projects, services, and writings. Built with a focus on aesthetics, speed, and user experience.

## üöÄ Features

*   **Responsive Design**: Fully optimized for all devices, from mobile phones to large desktop screens.
    *   *Smart Tables*: Pricing tables automatically transform into card layouts on mobile.
*   **Dark & Light Mode**: Seamless theme switching with persistent local storage preference.
*   **Professional Animations**:
    *   Cinematic "blur-to-focus" text reveals using **GSAP**.
    *   Smooth scroll animations via **AOS (Animate On Scroll)**.
    *   Interactive hover effects for navigation and cards.
*   **Dynamic Functionality**:
    *   **Currency Converter**: Real-time USD to PHP conversion for service rates using the ExchangeRate-API.
*   **Content Sections**:
    *   **Portfolio**: Detailed project showcases with case study layouts.
    *   **Writings**: A tabbed section for Articles and Blogs.
    *   **Rate**: Clear service pricing tiers.
    *   **About**: Professional biography and tech stack overview.
