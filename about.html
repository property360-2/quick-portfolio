About Me

Beyond the Code

Hi! I’m Dionisio Alvior aka Jun, a 21-year-old full stack software developer and a 4th-year BSIS student. I love building web applications that are functional, beautiful, and user-friendly, using a wide variety of tech stacks like Python (Django + Templates), React, Next.js, Node.js (Prisma), and PHP (Laravel & vanilla PHP).
